 テーマ「bootstrap2016」2016/06/23版
====================================

------------------------------------
 テーマフォルダ直下のテンプレート
------------------------------------
■ 404.html
404ページで表示されるテンプレートです。

■ base.html
ベースとなるパーツを集めたテンプレートです。

■ entry.html
詳細ページのテンプレートです。

■ index.html
一覧ページのテンプレートです。

■ template.yaml
bootstrap2016テーマを選択したときに、テンプレートをまとめて設定できるファイルです。使用するかどうかは管理画面＞コンフィグ＞テーマ設定 にて変更できます。

■ top.html
トップページのテンプレートです。

■ varstable.html
テンプレートの変数化で記述されているテンプレートたちを一覧にまとめたテンプレートです。

------------------------------------
フォルダ構成
------------------------------------
■ admin
bootstrap2016テーマオリジナルの管理画面のテンプレートが入ったフォルダ。
bootstrap2016テーマでは主に以下のテンプレートが入っています。
・ カスタムフィールド（ブログ・カテゴリー・エントリー・モジュール）
・ ダッシュボード

■ カテゴリーコード名フォルダ
a-blog cmsで作ったカテゴリーと同じコード名でフォルダを作ると、その中にあるindex.htmlとentry.htmlはそのカテゴリーの専用テンプレートとなります。
・ contact（お問い合わせ）
・ news（お知らせ）
・ realestate（物件情報）

■ css
bootstrap2016テーマで使っているCSSが入ったフォルダです。

■ fonts
Bootstrapで同梱されているGlyphiconsのフォントが入ったフォルダです。

■ images
bootstrap2016テーマで使っている画像が入ったフォルダです

■ include
bootstrap2016テーマ内のインクルードしているテンプレートをまとめたフォルダです。

■ js
bootstrap2016テーマで使っているJavaScriptが入ったフォルダです。

■ scss
bootstrap2016テーマで使っているSCSSファイルが入っているフォルダです。このscssフォルダに入っているSCSSファイルをコンパイルしてCSSを出力しています。

■ vars
bootstrap2016テーマで使っているsetTemplateやsetRenderedのテンプレートが入ったフォルダです。
以下のようなファイルが入っています。
- all.html
- head.html
- m_category.html
- m_entry.html
- m_siteparts.html
- m_tag.html
ファイル名の「m_」はモジュールの意味で、モジュールのカテゴリーごとにファイルが分けられています。
all.htmlではすべてのテンプレートの変数化用のテンプレートが読み込まれています。
all.html自体は/include/body_start.htmlにて読み込まれており、全テンプレートで使用できるようになっています。

------------------------------------
 a-blog cmsの基本機能
------------------------------------
■ カスタムフィールド

ブログ
・ サイトの設定
・ SEOの設定
・ OGPの設定
・ 会社情報
・ SNSアカウントの情報
・ アクセス解析

カテゴリー
・ SEOの設定
・ ページタイトルの設定


エントリー
・ ピックアップのチェックボックス
・ 物件情報でカスタムフィールドのサンプル（※ 物件情報のみ）

モジュール
・ 見出しの設定


■ フォーム
・ お問い合わせのページにてお問い合わせフォーム
・ すべてのページにて検索フォーム

■ エントリーの固定表示
エントリー編集ページの詳細設定の中にあるファイル名の入力欄を空にすると、エントリーが固定表示される機能。会社概要のページで実装しています。

■ カスタムフィールドを使ったモジュールの絞り込み表示
・ サイドカラムのピックアップで実装。あらかじめ各エントリーにカスタムフィールドを実装し、管理画面のモジュールの「フィールド」の設定にて、条件を絞り込んで一致したエントリーを表示しています。
