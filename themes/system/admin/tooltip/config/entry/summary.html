表示順
どのようなルールに従って表示されるかの設定です。
表示件数
一度に表示される件数です。
オフセット
表示順で並び替えた後の先頭から数えて、設定した件数のエントリーを表示しなくします。
新着期間
エントリーの公開日時を基点に、ここで設定した期間中はnewブロックが出力されます。
前後に表示するページ数
ページャーの設定です。表示中のページを基点として、ここで設定した値の数だけページ番号が表示されます。
現在のページの要素の属性
ページャーの設定です。表示中のページ番号に割り当てられる属性です。(変数名 : pageCurAttr)
インデキシング
チェックを外すと、インデキシングされていないエントリーも表示されます。
シークレットエントリー
チェックをすると、ログインしていなくてもシークレットブログのエントリーが表示されます。
シークレットな子ブログのエントリーを出すためには、モジュールIDの設定でブログIDは指定せず、ブログの階層は下階層のブログを含めるようにしてください。
ユニット
指定した件数のエントリーごとに、unitブロックが表示されます。何件ごとに折り返すなどの指定に応用できます。
エントリーがない場合の表示
チェックすると、指定条件でエントリーが見つからなかった場合にnotFoundブロックを出力します。
エントリーがない場合のHTTP応答
チェックをすると、このモジュールが表示すべきエントリー情報が0件のときに、リクエストされたページの応答を404 Not Foundとします。
ノーイメージ
チェックを外すと、画像がないエントリーを表示しなくなります。
サイズ
画像のサイズを指定する設定です。
トリミング
サイズで指定された大きさに、画像をトリミングして表示します。
拡大　( トリミングがoffの場合のみ )
画像が指定されたサイズよりも小さいときに、指定されたサイズまで拡大します。
表示位置 ( トリミングがoffの場合のみ )
画像を指定されたサイズの中心に表示します。
投稿日時・更新日時
チェックをすると投稿日時・更新日時を出力できるようになります。
サマリー
チェックをするとサマリーを出力できるようになります。
タグ
チェックをするとタグを出力できるようになります。
メインイメージ
チェックをするとメインイメージを出力できるようになります。
ページャー
チェックをするとページャー機能を使用することができます。
フィールド基本設定
チェックをすると各フィールドの基本設定を出力することができます。
カスタムフィールド
チェックをすると各フィールドのカスタムフィールドを出力することができます。
詳細のエントリー表示
チェックをすると詳細ページでそのページのエントリーを除いたものが出力されます。
