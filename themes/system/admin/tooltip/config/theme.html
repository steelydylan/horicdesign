テーマ ディレクトリ名
適用されるテーマのディレクトリ名です。「a-blog cms設置ディレクトリ/themes/」以下のディレクトリになります。
テンプレート設定
チェックがついている場合テーマディレクトリ直下のtpl.yamlファイルのテンプレートを優先して読む込みます。（template.yamlは継承されます）
トップページ
そのサイト(ブログ)のトップページにアクセスした際に適用されるテンプレートファイルです。
一覧ページ
複数のエントリーを一覧表示する際に適用されるテンプレートファイルです。
詳細ページ
個別のエントリーを表示する際に適用されるテンプレートファイルです。
エラーページ(404 Not Found)
存在しないURLにアクセスした際に適用されるテンプレートファイルです。
管理ページ
管理ページ(このページなど)を表示する際に適用されるテンプレートファイルです。
エントリー編集ページ
エントリーを投稿・編集する際に適用されるテンプレートファイルです。
ユニット追加ページ
エントリーにユニットを追加する際に適用されるテンプレートファイルです。
ログインページ
「設置URL/login/」で表示されるログインページに適用されるテンプレートファイルです。
