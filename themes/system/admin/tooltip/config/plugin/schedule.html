参照先スケジュール
データ参照先のスケジュールセット名を選択します。
デフォルトの表示
URLコンキストに年月の情報が乗っていない状態の表示単位を設定します。
年の表示モード
年表示時のリスト表示か七曜表（カレンダー）表示を選択します。
月の表示モード
月表示時のリスト表示か七曜表（カレンダー）表示を選択します。
指定日数の表示モード
指定日数表示時の表示です。現在はリスト表示のみ対応しています。
月表示のときの前後月の表示
月表示時に、当月に加えて前後指定月分を表示できます。
指定日数表示のときの前後月の表示
指定日数表示時に、当日に加えて前後の指定日数分を表示できます。
ユニット
指定した件数の月ごとに、unitブロックが表示されます。何件ごとに折り返すなどの指定に応用できます。
年の表示形式
年の表示形式を選択します。
月の表示形式
月の表示形式を選択します。
日の表示形式
日の表示形式を選択します。
曜日の表示形式
曜日の表示形式を選択します。
ラベルを追加する
ラベルを追加します。ラベルはリスト表示モードで出力されます。
ラベルを編集する
登録済みのラベルを編集します。削除にチェックしたラベルは「ラベルを編集する」を押したときにまとめて削除されます。
