モジュール
IDを付与する対象のモジュールです。
ID
モジュールに付与するID名です。これと同じ記述がテンプレート側にも必要になります。(半角英数字)
名前
このモジュールIDを管理するための名前です。日本語可。
説明
このモジュールIDがどのような動作をするか、などのメモです。
ブログID（bid）
このモジュールIDの表示内容を、ブログの管理ID(番号)で指定します。
ユーザーID（uid）
このモジュールIDの表示内容を、ユーザーの管理ID(番号)で指定します。
カテゴリーID（cid）
このモジュールIDの表示内容を、カテゴリーの管理ID(番号)で指定します。
エントリーID（eid）
このモジュールIDの表示内容を、エントリーの管理ID(番号)で指定します。
キーワード（keyword）
このモジュールIDの表示内容を、エントリーについてキーワードで指定します。
タグ（tag）
このモジュールIDの表示内容を、エントリーについてタグで指定します。
フィールド（field）
このモジュールIDの表示内容を、カスタムフィールドで指定します。
開始日時（start）
エントリーの日時に対して表示期間を指定します。ここで指定した日時以降のエントリーが表示されます。
終了日時（end）
エントリーの日時に対して表示期間を指定します。ここで指定した日時以前のエントリーが表示されます。
ページ（page）
このモジュールIDの表示内容を、ページ番号で指定します。
ページ（page）
このモジュールIDの表示内容を、表示順で指定します。
ブログ
下階層のブログを表示内容に含めるかの設定です。
カテゴリー
下階層のカテゴリーを表示内容に含めるかの設定です。
